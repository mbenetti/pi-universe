# Levenshtein (1966) — Binary Codes Capable of Correcting Deletions, Insertions, and Reversals

## Citation

> Levenshtein, V. I. (1966). Binary codes capable of correcting deletions, insertions, and reversals. *Soviet Physics — Doklady*, **10**(8), 707–710.
>
> **DOI:** [10.1007/978-3-662-02708-4_58](https://doi.org/10.1007/978-3-662-02708-4_58) (reprint in *String Transductions and Their Applications*, 2004)

## Summary

This seminal paper introduces what is now universally known as **Levenshtein distance** — the minimum number of single-character edit operations (insertions, deletions, and substitutions) required to transform one string into another. Levenshtein frames the problem in the context of error-correcting codes over channels subject to insertion and deletion noise, establishing both theoretical bounds on code distance and a dynamic-programping recurrence for computing edit distance.

**Key contributions:**

- Formal definition of edit distance under insertion, deletion, and reversal operations.
- Derivation of a dynamic programming recurrence that runs in O(nm) time for strings of length n and m — the foundational algorithm still taught today.
- Theoretical bounds on the minimum Hamming distance of codes under edit operations, establishing the edit distance as a metric for code design.
- Introduction of the concept that the "distance" between two strings can be measured by the cost of transforming one into the other.

## Relevance to Distance Compression Under Probabilistic Perturbation

Levenshtein's work provides the **foundational distance metric** upon which distance compression research is built. When studying how probabilistic perturbations (random insertions, deletions, substitutions) affect string similarity, the Levenshtein distance serves as the canonical ground-truth measure. Understanding its computational structure — particularly the DP recurrence — is essential for analyzing how compressed representations of the distance matrix behave under perturbation. Any compression scheme for edit distance must be consistent with Levenshtein's metric properties (triangle inequality, symmetry, identity), making this paper the starting point for the entire research area.
