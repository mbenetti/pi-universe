# Cassis, Kociumaka & Wellnitz (2023) — Optimal Algorithms for Bounded Weighted Edit Distance

## Citation

> Cassis, B., Kociumaka, T., & Wellnitz, S. (2023). Optimal algorithms for bounded weighted edit distance. In *Proceedings of the 55th Annual ACM SIGACT Symposium on Theory of Computing (STOC 2023)*, 1596–1609.
>
> **DOI:** [10.1145/3564246.3585203](https://doi.org/10.1145/3564246.3585203)

## Summary

This recent STOC paper presents the first asymptotically optimal algorithms for computing bounded weighted edit distance under the Strong Exponential Time Hypothesis (SETH). The authors resolve a long-standing open problem by showing that weighted edit distance can be computed in O(k · n) time (where k is the distance bound and n is the string length) under certain conditions, matching the theoretical lower bound.

**Key contributions:**

- **Optimal time complexity**: Matching O(kn) upper and lower bounds for bounded weighted edit distance, resolving the complexity question up to the SETH assumption.
- **Weighted operation costs**: Extends earlier unweighted results to the general weighted case, where insertions, deletions, and substitutions may have arbitrary positive costs.
- **Tight connection to SETH**: Establishes that improving beyond O(kn) would refute SETH, providing a fundamental complexity barrier.
- Novel algorithmic techniques combining DP pruning with bit-parallelism and window-based approaches.

## Relevance to Distance Compression Under Probabilistic Perturbation

This paper establishes the **theoretical complexity ceiling** for edit distance computation, which directly informs what compression schemes can hope to achieve. If the optimal algorithm for computing bounded edit distance requires Ω(kn) time, then any lossless compression of the full edit distance matrix cannot avoid this fundamental cost in the general case. For distance compression under probabilistic perturbation, this result is significant because:

1. It quantifies the **information-theoretic lower bound** — you cannot compress below the inherent complexity of the distance computation.
2. The bounded-distance setting (threshold k) is precisely the regime where probabilistic perturbation matters most, since perturbations naturally produce bounded deviations.
3. The weighted case directly models non-uniform perturbation probabilities, making this the closest theoretical framework to the distance compression problem.
